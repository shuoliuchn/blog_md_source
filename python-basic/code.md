## 编码初识

常见编码集：

1. ascii：
   - 不支持中文
   - 一个字符占用 8 位
2. gbk（包含 ascii）国标码：
   - 一个英文字符占用 8 位（1 字节）
   - 一个中文字符占用 16 位（2 字节）
3. Unicode：
   - 英文：4 个字节，32 位
   - 中文：4 个字节，32 位
4. utf-8（最流行的编码集）：
   - 英文：1 个字节，8 位
   - 欧洲：2 个字节，16 位
   - 亚洲：3 个字节，24 位

单位转换：

- 1 字节 = 8 位
- 1 Byte = 8 bits
- 1024 bytes = 1 KB
- 1024 KB = 1 MB
- 1024 MB = 1 GB
- 1024 GB = 1 TB    # 够用了
- 1024 TB = 1 PB
